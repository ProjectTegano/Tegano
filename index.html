Project Wolframe

A flexible client-server ecosystem for business applications. Just configure it to get a powerful ERP or CRM system, or whatever else your business needs. Use the C++ framework to build your own professional client-server applications.

This the git repository of Project Wolframe.

Find out more about Project Wolframe at wolframe.org.

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/mbarbos/Wolframe
